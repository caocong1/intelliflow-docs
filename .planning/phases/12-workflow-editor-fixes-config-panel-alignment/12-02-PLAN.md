Create 8 new components following the architecture in RESEARCH.md:

**FlowViewport.tsx**: Simple wrapper div with `transform: translate(panX, panY) scale(zoom)` and `transform-origin: 0 0`. Takes panX, panY, zoom, children as props.

**FlowBackground.tsx**: SVG pattern background (dot grid). Adjusts dot positions based on viewport offset so dots appear to move with pan. Takes viewport as prop.

**FlowControls.tsx**: Fixed-position panel with zoom in (+10%), zoom out (-10%), and fit-view buttons. Takes viewport/setViewport and a fitView callback as props. Style with Tailwind — small floating panel in bottom-left.

**NodeHandle.tsx**: Small circle div (12x12px) positioned on node border. Props: type ("source"|"target"), position (HandlePosition), onMouseDown callback. Styled with Tailwind: `w-3 h-3 rounded-full bg-gray-400 hover:bg-indigo-500 absolute`. Position calculation uses the `position` prop to set top/left/right/bottom CSS.

**FlowNode.tsx**: Wrapper for each node. Absolutely positioned at `node.position.x/y`. Responsibilities:
1. Absolute positioning via inline style
2. Mouse down → start drag (set window mousemove/mouseup listeners, stop propagation to prevent pan)
3. ResizeObserver on the node div → call `updateNodeSize` on the store
4. Render source and target NodeHandle components
5. Render children (the actual node content component)
6. Selected state: add indigo ring/border when selected
7. On drag, compute delta in flow coordinates and call `updateNodePosition`

**EdgeRenderer.tsx**: SVG `<g>` containing:
1. Arrow marker in `<defs>`
2. Invisible wide (12px) hit-area path with `pointer-events: stroke` and cursor pointer
3. Visible path (2px normal, 3px selected) with indigo color and `marker-end` arrow
4. Uses createMemo to compute path data from source/target positions + handle positions
5. Takes edge, sourcePos, targetPos, sourceHandlePos, targetHandlePos, selected, onSelect as props

**TempEdge.tsx**: SVG path rendered during connection creation. Takes sourcePos and mousePos (both in flow coords). Renders a dashed bezier from source to mouse. Only rendered when a connection drag is active.

**FlowCanvas.tsx**: The main orchestrator component. Architecture:
```
<div class="relative w-full h-full overflow-hidden" tabIndex={0} ref={canvasRef}
     onWheel={handleWheel} onMouseDown={handleCanvasMouseDown}>
  <FlowBackground viewport={viewport()} />
  <FlowViewport panX={viewport().x} panY={viewport().y} zoom={viewport().zoom}>
    {/* SVG layer */}
    <svg style="position:absolute; top:0; left:0; width:100%; height:100%; pointer-events:none; overflow:visible">
      <For each={edges}>{edge => <EdgeRenderer ... />}</For>
      <Show when={connecting()}><TempEdge ... /></Show>
    </svg>
    {/* HTML node layer */}
    <For each={nodes}>{(node, i) =>
      <FlowNode node={node} selected={...} onDragStart={...} onHandleMouseDown={...}>
        <Switch><Match when={node.type === "input_transform"}><InputTransformNode ... /></Match>
        ... (all 5 types)
        </Switch>
      </FlowNode>
    }</For>
  </FlowViewport>
  <FlowControls viewport={viewport()} setViewport={setViewport} fitView={fitView} />
</div>
```

Event handling in FlowCanvas:
- **Wheel**: Zoom toward mouse position (see RESEARCH.md Pattern 2). Clamp zoom 0.2-4.
- **MouseDown on empty space**: Start pan. Set window mousemove (update viewport x/y) and mouseup (stop pan) listeners.
- **MouseDown on node (via FlowNode stopPropagation)**: Start node drag. Handled by FlowNode.
- **MouseDown on handle (via NodeHandle)**: Start connection. Store `{ sourceNodeId, sourceHandlePosition, mousePos }` in a `connecting` signal. Window mousemove updates mousePos. Window mouseup on a target handle creates edge (via store.addEdge); on empty space cancels.
- **Drop from library**: Listen for `dragover` (preventDefault) and `drop` events. Read `application/solid-flow-node` from dataTransfer (matching NodeLibraryPanel's drag format). Convert drop screen position to flow coordinates. Create new node at that position with default config. Auto-connect: find the nearest node with no outgoing edge and create an edge to the new node.
- **fitView**: Compute bounding box of all nodes, then set viewport to center and scale all nodes into view with padding.

Props for FlowCanvas: receive the flow store (nodes, edges, setters, helpers) and callbacks for node selection, edge selection, config changes.
