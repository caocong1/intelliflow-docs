  <action>
    **auth.guard.ts — Auth middleware using ElysiaJS guard + derive pattern:**
    - `authPlugin`: new Elysia({ name: 'auth' }), use @elysiajs/jwt plugin with JWT_SECRET from env, derive async handler that reads JWT from cookie named 'token', verifies it, returns { user: payload | null } in context. Set cookie config: httpOnly true, secure based on NODE_ENV, sameSite 'lax', path '/'.
    - `requireAuth`: new Elysia({ name: 'requireAuth' }), use authPlugin, guard with beforeHandle — if !user, return 401 { error: 'Unauthorized' }.
    - `requireAdmin`: new Elysia({ name: 'requireAdmin' }), use authPlugin, guard with beforeHandle — if !user return 401, if user.role !== 'admin' return 403 { error: 'Forbidden' }.
    - Export all three: authPlugin, requireAuth, requireAdmin.

    **auth.service.ts — Auth business logic:**
    - `validateCredentials(username: string, password: string)`: Query users table by username, check isActive, verify password with Bun.password.verify. Return user object (id, username, displayName, role) on success, null on failure.
    - `getUserById(id: string)`: Query users table by id, return user object (id, username, displayName, role, isActive) or null.

    **auth.routes.ts — Auth endpoints:**
    - POST /login: Accept { username, password } (validate with t.Object). Call validateCredentials. On failure return 401. On success, sign JWT with { sub: user.id, role: user.role, exp: 7 days }. Set JWT in httpOnly cookie named 'token' (maxAge 7 days, httpOnly, secure in prod, sameSite lax, path /). Return { user: { id, username, displayName, role } }.
    - GET /me: Use authPlugin derive. If no user in context, return { user: null }. Otherwise call getUserById(user.sub), return { user } or { user: null }.
    - POST /logout: Clear the token cookie by setting it to empty with maxAge 0. Return { success: true }.
    - All routes under prefix '/auth'.

    **Update index.ts:**
    - Import authRoutes from modules/auth/auth.routes
    - Add .use(authRoutes) to the Elysia app
    - Ensure `export type App = typeof app` is AFTER all .use() calls so Eden Treaty captures all routes
  </action>
