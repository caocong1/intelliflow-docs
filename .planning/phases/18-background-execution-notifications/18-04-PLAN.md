**B) Create `NotificationBell.tsx`**:

A small bell icon component for the sidebar:

```typescript
// Props: onOpen: () => void (toggles drawer)
// Internal state:
const [unreadCount, setUnreadCount] = createSignal(0);

// Poll unread count every 15 seconds
const NOTIFY_POLL_INTERVAL = 15;
const pollTimer = setInterval(async () => {
  const { count } = await getUnreadCount();
  setUnreadCount(count);
}, NOTIFY_POLL_INTERVAL * 1000);
// Also fetch immediately on mount
onMount(() => getUnreadCount().then(r => setUnreadCount(r.count)));
onCleanup(() => clearInterval(pollTimer));
```

Render:
- Bell SVG icon (use a simple bell icon from heroicons or inline SVG)
- If unreadCount > 0: show a small red circle with the count (positioned top-right of bell icon)
- `cursor-pointer` class, onClick calls `props.onOpen()`
- Use `relative` positioning for the badge overlay
