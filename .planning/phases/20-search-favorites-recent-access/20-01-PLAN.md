Create `packages/backend/src/modules/user-activity/` directory with two files.

**user-activity.service.ts** — Five exported functions:

1. `toggleFavorite(userId, targetType, targetId)`: Check if favorite exists (SELECT WHERE userId + targetType + targetId). If exists, DELETE it and return `{ favorited: false }`. If not, INSERT and return `{ favorited: true }`.

2. `listFavorites(userId)`: SELECT from userFavorites WHERE userId, ORDER BY createdAt DESC. For each record, LEFT JOIN to the target table based on targetType to get name/title. Return grouped by type: `{ projects: [...], documents: [...], workflows: [...] }`. Filter out records where target entity no longer exists (deleted/null join). Each item includes: id (favorite id), targetId, targetType, name (project.name or document.title or workflow.name), createdAt.

3. `checkFavorites(userId, items: Array<{targetType, targetId}>)`: Given a list of target items, return which ones are favorited. SELECT from userFavorites WHERE userId AND (targetType, targetId) IN items. Return a Set or array of `targetType:targetId` strings. This is for the frontend to show filled/empty stars.

4. `recordAccess(userId, targetType, targetId)`: UPSERT into userRecentAccess — use `.onConflictDoUpdate({ target: [userRecentAccess.userId, userRecentAccess.targetType, userRecentAccess.targetId], set: { accessedAt: new Date() } })`. Then evict oldest: SELECT id FROM userRecentAccess WHERE userId ORDER BY accessedAt DESC OFFSET 20, then DELETE those ids if any.

5. `listRecentAccess(userId, limit = 20)`: SELECT from userRecentAccess WHERE userId ORDER BY accessedAt DESC LIMIT limit. LEFT JOIN to target tables based on targetType for name/title. Filter out null joins (deleted targets). Each item includes: id, targetId, targetType, name, accessedAt.

**user-activity.routes.ts** — Elysia route with prefix "/user-activity":
- Use `requireAuth` guard.
- `POST /favorites/toggle` — body: `{ targetType: t.Union([t.Literal("project"), t.Literal("document"), t.Literal("workflow")]), targetId: t.String() }`. Calls toggleFavorite.
- `GET /favorites` — Calls listFavorites(user.id).
- `POST /favorites/check` — body: `{ items: t.Array(t.Object({ targetType: ..., targetId: t.String() })) }`. Calls checkFavorites. Used by frontend to batch-check favorite status.
- `POST /recent-access` — body: `{ targetType: ..., targetId: t.String() }`. Calls recordAccess.
- `GET /recent-access` — optional query param `limit`. Calls listRecentAccess.
