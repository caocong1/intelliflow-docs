2. Define `ALLOWED_TAGS` as a Set: ['p','br','strong','em','code','pre','h1','h2','h3','h4','h5','h6','ul','ol','li','a','span','div','blockquote','hr'].
3. Define `ALLOWED_ATTR` as a Set: ['href','title','class','data-var'] (covers all attributes used in render-markdown.tsx inlineFormat output and PromptEditor.tsx buildEditorHTML).

5. Export `export function sanitizeHtml(html: string): string { return DOMPurify.sanitize(html, { RETURN_TRUSTED_TYPE: false }); }`.
6. Add a JSDoc comment: `/** * Sanitize untrusted HTML by removing dangerous tags (script, iframe, etc.), * attributes (onerror, onclick, javascript: href), and SVG/mMath content. * Uses a conservative allowlist of markdown-friendly tags and safe attributes. */`.
Note: The `RETURN_TRUSTED_TYPE` option is set to false to return plain strings (not TrustedHTML) to avoid TypeScript compatibility issues with SolidJS innerHTML prop.
</action>
