**Extend Props interface** to accept AI editing context:

```typescript
interface Props {
  content: string;
  onChange: (content: string) => void;
  readOnly: boolean;
  placeholder?: string;
  // New: AI editing context
  documentId?: string;
  nodeExecutionId?: string;
  nodes?: NodeExecution[];        // DocumentRuntimeState.nodes for security context
  currentNodeIndex?: number;      // Index of current node in nodes array
  availableModels?: Array<{ id: string; name: string; deploymentType: "cloud" | "local" }>;
  defaultModelId?: string;        // First model from node config
}
```

**Add AI editing state machine** inside the component:

Signals:
- `aiEditState`: `"idle" | "streaming" | "diff_preview"` (createSignal, default "idle")
- `originalSelection`: `{ text: string; start: number; end: number } | null`
- `streamingContent`: accumulated SSE delta text (string signal)
- `completedContent`: final AI result after streaming completes (string signal)
- `selectedModelId`: current model selection (initialized from `props.defaultModelId`)
- `abortController`: `AbortController | null` for cancellation

**Derived signals:**
- `isPostRestore`: computed from `props.nodes` and `props.currentNodeIndex` -- check if any preceding node has `nodeType === "restore"` and `status === "completed"`. Use `createMemo`.
- `filteredModels`: if `isPostRestore()` is true, filter `props.availableModels` to `deploymentType === "local"` only. Otherwise return all models.
- Validate `selectedModelId` against `filteredModels` -- if current selection not in filtered list, fall back to first available model.

**Action handler `handleAIAction(action, customInstruction?)`:**
1. Save current selection to `originalSelection` signal
2. Set `aiEditState` to "streaming"
3. Set `streamingContent` to ""
4. Create new AbortController
5. Call `streamSSE` with:
   - url: `/api/runtime/${props.documentId}/inline-edit/${props.nodeExecutionId}/stream`
   - method: "POST"
   - body: `{ action, selectedText: originalSelection.text, modelId: selectedModelId(), customInstruction }`
   - onDelta: append to `streamingContent`
   - onComplete: set `completedContent`, transition to "diff_preview" state
   - onError: show error toast, restore original, transition to "idle"
   - signal: abortController.signal

**Cancel handler:**
1. Call `abortController?.abort()`
2. Restore original text in editor content
3. Set `aiEditState` to "idle"

**Accept handler:**
1. Replace the selected range in editor content with `completedContent()`
2. Call `props.onChange(newContent)`
3. Reset all AI editing state to idle

**Reject handler:**
1. Restore original text (no-op since we only replace on accept)
2. Reset `aiEditState` to "idle"
3. Keep toolbar visible for retry (selection stays stored in `originalSelection`)
