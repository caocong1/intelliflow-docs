  <action>Create `packages/backend/src/common/sanitize.ts`. Implement `export function sanitizeFilename(input: string): string` as a pure function with no Node.js dependencies. The function must:
1. Strip all null bytes (\x00) from the input string (rejecting them entirely).
2. Remove any path separators (/ and \) so only a bare basename remains.
3. Strip all leading dot characters from the resulting basename (prevents .bashrc style hidden files).
4. If the result is empty or all whitespace, return a default string "uploaded_file".
5. Replace any remaining whitespace characters with underscores.
6. Return the sanitized result as a string (never a path).

Example mappings:
- "../etc/passwd" -> "etc_passwd"
- ".hidden" -> "hidden"
- "file\x00name.txt" -> "filename_txt"
- "normal file.pdf" -> "normal_file_pdf"
- "\x00.txt" -> "uploaded_file"
