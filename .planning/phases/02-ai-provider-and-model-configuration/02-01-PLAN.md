**Create `packages/backend/src/modules/providers/providers.service.ts`** following the document-types service pattern:

1. `listProviders()` — SELECT all providers ordered by createdAt desc. For each provider, mask apiKey (show only last 6 chars: `sk-...${key.slice(-6)}`). Return array of providers.
2. `createProvider(data: { name, type?, baseUrl, apiKey })` — Strip trailing slashes from baseUrl (`baseUrl.replace(/\/+$/, "")`). INSERT and return the created provider (with masked key).
3. `updateProvider(id, data: { name?, baseUrl?, apiKey? })` — If baseUrl provided, strip trailing slashes. If apiKey is empty string or undefined, do NOT update the key (keep existing). PATCH only provided fields using Drizzle `set()`. Set `updatedAt` to now. Return updated provider (masked key). Throw if not found.
4. `deleteProvider(id)` — Check if provider has any models (SELECT count from models where providerId = id). If has models, throw error "请先删除该供应商下的所有模型" (block deletion). Otherwise DELETE provider. Throw if not found.
5. `toggleProviderStatus(id)` — In a single transaction:
   - Fetch current provider
   - Flip `isActive`
   - If disabling (isActive was true → now false): UPDATE all models under this provider SET `isProviderDisabled = true`
   - If enabling (isActive was false → now true): UPDATE all models under this provider SET `isProviderDisabled = false`
   - Return updated provider (masked key)
6. `testProviderConnection(id, modelId?: string)` — Fetch provider (full apiKey from DB). Use native `fetch` to POST to `${provider.baseUrl}/chat/completions` with:
   - Headers: `Content-Type: application/json`, `Authorization: Bearer ${apiKey}`
   - Body: `{ model: modelId || "doubao-seed-2.0-lite", messages: [{ role: "user", content: "hi" }], max_tokens: 5 }`
   - Use `AbortSignal.timeout(15000)` for 15s timeout
   - Measure latency with `Date.now()` before/after
   - Return `{ success, message, latencyMs }` — on success: message "连接成功"; on HTTP error: message with status code and truncated body; on timeout: "连接超时（15秒）"; on other error: error message

Helper function `maskApiKey(key: string): string` — returns `"sk-...${key.slice(-6)}"` or if key length < 10 returns `"***"`.
