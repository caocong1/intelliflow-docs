**3. Update `FlowCanvas.tsx`** — Integrate selection and deletion:

**Selection integration:**
- Create a selection store via `createSelectionStore()`
- On node click (from FlowNode callback): call `selectNode(id, e.ctrlKey || e.metaKey || e.shiftKey)`
- On edge click (from EdgeRenderer callback): call `selectEdge(id)`
- On canvas mousedown (empty space, left button): if no modifier, start rubber-band selection. Track start position in flow coords. On mousemove, render SelectionBox in SVG layer. On mouseup, call `selectNodesInRect` with the bounding rect and clear the rubber-band.
- On canvas mousedown (empty space, left button, no rubber-band intent): also support pan. Use a brief threshold — if mouse moves >5px without releasing, it's a pan; if mouse stays still and releases, it's a deselect-all click. OR: use middle mouse button / Space+drag for pan, left click for selection. Recommendation: left click on empty space = rubber-band if dragged, deselect-all if clicked without drag. Middle mouse / Space+drag = pan.
- Pass `selected` boolean to each FlowNode and EdgeRenderer

**Deletion with confirmation:**
- Add `onKeyDown` handler to the canvas div (needs `tabIndex={0}` for focus)
- On Delete or Backspace key:
  - If any nodes or edges are selected, show a styled confirmation dialog (not browser `confirm()` — use a SolidJS `Show` with a modal overlay)
  - Confirmation message: `确认删除 ${nodeCount} 个节点和 ${edgeCount} 条连接？`
  - On confirm: call `store.removeNodes(selectedNodeIds())` and `store.removeEdges(selectedEdgeIds())`, then `clearSelection()`
  - On cancel: close dialog

**Batch node drag:**

- In FlowNode's drag handler, check if the node is in `selectedNodeIds`. If yes, compute delta and apply `updateNodePosition` to every selected node.
