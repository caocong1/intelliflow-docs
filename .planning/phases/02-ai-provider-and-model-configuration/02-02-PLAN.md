<!-- Backend API endpoints created by Plan 01 -->

Provider endpoints (all require admin auth):
- GET /api/providers — returns Provider[] with masked API keys
- POST /api/providers — body: { name, baseUrl, apiKey, type? } — returns created Provider
- PATCH /api/providers/:id — body: { name?, baseUrl?, apiKey? } — returns updated Provider
- DELETE /api/providers/:id — blocks if has models, returns error
- PATCH /api/providers/:id/status — toggles isActive with cascade to models
- POST /api/providers/:id/test — body: { modelId? } — returns { success, message, latencyMs }

Model endpoints (all require admin auth):
- GET /api/models/by-provider/:providerId — returns Model[]
- POST /api/models — body: { providerId, modelId, displayName, deploymentType? }
- PATCH /api/models/:id — body: { modelId?, displayName?, deploymentType? }
- DELETE /api/models/:id
- PATCH /api/models/:id/status — toggles model isActive
