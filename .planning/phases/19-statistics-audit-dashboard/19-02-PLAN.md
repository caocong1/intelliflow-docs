2. Create `src/lib/echarts.ts` — tree-shakeable ECharts setup:
   ```typescript
   import * as echarts from "echarts/core";
   import { BarChart, LineChart, PieChart } from "echarts/charts";
   import { GridComponent, TooltipComponent, LegendComponent, DatasetComponent } from "echarts/components";
   import { CanvasRenderer } from "echarts/renderers";

   echarts.use([BarChart, LineChart, PieChart, GridComponent, TooltipComponent, LegendComponent, DatasetComponent, CanvasRenderer]);

   export { echarts };
   export type { EChartsCoreOption } from "echarts/core";
   ```

3. Create `src/components/charts/ChartContainer.tsx` — reusable SolidJS chart wrapper:
   - Props: `class?: string`, `option: () => EChartsCoreOption`
   - Uses `let el: HTMLDivElement` ref
   - `onMount`: init echarts on el, set initial option, create ResizeObserver
   - `createEffect`: reactively update option via `chart.setOption(props.option(), true)`
   - `onCleanup`: disconnect ResizeObserver, dispose chart
   - Default class: `"w-full h-64"`
